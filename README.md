# Elad Tzalik - Personal Website

This is the source code for my personal academic website.

## About

I am a PhD student at the Mathematics Department at the Weizmann Institute of Science, advised by Prof. Itai Benjamini.

My research interests include Algorithms, Combinatorics, and High Dimensional Combinatorics and Geometry.

## Files

- `index.html` - Main HTML file
- `styles.css` - CSS styling
- `script.js` - JavaScript for smooth scrolling and navigation
- `portrait.png` - Profile photo

## Website Features

- Clean, academic-style design
- Responsive layout (mobile-friendly)
- Smooth scrolling navigation
- Publications list with links to arXiv and conference proceedings
- Contact information

## Updating the Website

To update the website after making changes:

```bash
git add .
git commit -m "Description of changes"
git push
```

GitHub Pages will automatically rebuild the site within a few minutes.

- HTML5
